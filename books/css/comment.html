スタイルシートリファレンス

//一行コメントもしくは/*複数行コメント*/･････CSSソース中にコメントを入れる

コメント部分はブラウザには無視されるので、一時的にCSSを無効にしたりCSSソース内にメモを残す際などに使用します。
コメントを入れ子にすることはできません。


値

//一行コメント
//の後ろの文字は一行コメントになります
/* 複数行コメント */
/* */で囲んだ範囲は複数行コメントになります。
